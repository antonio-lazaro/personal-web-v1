## Set Up

1. Install the Gatsby CLI.

   ```shell
   npm install -g gatsby-cli
   ```

2. Install the correct version of Node using NVM.

   ```shell
   nvm install
   ```

3. Install dependencies.

   ```shell
   npm install
   ```

4. Start development server.

   ```shell
   npm start
   ```
